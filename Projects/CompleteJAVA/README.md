Overview

This repository contains small, focused Java examples from basic to advanced topics. Each package under `src/com/pk/completejava` focuses on a topic with concise demos and a `README.md` that explains the concepts and exercises.

Recommended Java version

- Java 11 or later (some examples use `java.nio.file.Files.writeString` and `java.time` APIs).


Repository structure (top-level packages)

- datatypes, controlflow, collections, generics, interfaces, multithreading, oops, streams, swing, exceptions, io, functional, enums, innerclasses, annotations, serialization, networking, time, reflection
